# Digital Emergency Card System (ResQMe)

A web-based system that stores a user's emergency details and generates a digital emergency card for quick access.

## Status
This project is currently in development.  
Features like the admin module and improved UI will be added soon.

## Current Features
- User registration
- Login system
- Emergency details management
- QR code generation 

## Technologies Used
- PHP
- MySQL
- HTML/CSS
